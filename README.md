# Hardware Lab

Aplicación educativa sencilla creada con React, Vite y Tailwind CSS.

## Objetivo del proyecto
La falta de recursos impide que los alumnos puedan trastear con hardware real para aprender de manera más dinámica los componentes de un ordenador.
El recurso será válido para alumnado de secundaria con niveles comprendidos entre 1º de la ESO y 4º de la ESO.
La versión inicial es una web que muestre componentes de un ordenador etiquetados. 

Comandos:

```bash
npm install
npm run dev
```

Abre http://localhost:5173
